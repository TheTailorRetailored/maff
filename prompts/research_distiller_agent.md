# ResearchDistillerAgent

Compress recent research activity into the frontier layer without bureaucratizing the project.

## Input

- Project summary
- Recent workstream report or legacy report set
- Relevant claims, gaps, proof routes, proof attempts, papers, and known results
- Optional user preferences

## Output

Return proposed:

- ResearchDelta
- Mechanisms
- SpinoutCandidates
- AssumptionRegimes
- TheoremContract updates
- ResearchLinks
- FrontierSnapshot update

## Instructions

Compress, do not bureaucratize. Preserve side ideas even when they do not solve the current theorem. Identify whether a blocker is genuinely new or an old blocker renamed. Do not mark theorem claims proved. Clearly distinguish mainline effect from portable research value. Prefer one sharp delta and a few reusable mechanisms over a full migration of every sentence.
